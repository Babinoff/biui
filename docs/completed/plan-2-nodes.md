# План реализации Этапа 2: Нодовый движок (Node Engine)

## Цель этапа
Интеграция библиотеки для работы с нодами и создание базовой структуры узлов с портами ввода/вывода. Этот этап заложит основу для будущей логики обработки данных.

---

  - [x] React Flow (рекомендуется для MVP)
  - [ ] React DnD + кастомная реализация
  - [ ] Nodemailer или аналогичные решения
- [x] Установка выбранной библиотеки

  - [x] Базовый стиль фона (сетка/градиент)

- [x] Определить интерфейс `NodeData`:

  - [x] Заголовок/имя ноды
  - [x] Блок портов:

### Интерфейсы (предварительные)

```typescript
interface Port {
  id: string;
  type: 'input' | 'output';
  label?: string;
  isConnected: boolean;
}

interface Node {
  id: string;
  type: string;
  position: Position;
  data: NodeData;
  inputs: Port[];
  outputs: Port[];
}
```

### Интеграция React Flow (пример)

```tsx
import { ReactFlow, MiniMap, Controls } from 'react-flow-renderer';

const NodeEditor = () => {
  return (
    <ReactFlow nodes={nodes} edges={edges}>
      <MiniMap />
      <Controls />
    </ReactFlow>
  );
};
```
