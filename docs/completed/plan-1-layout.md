## Цель этапа

## Блок реализации: загрузка данных из CSV

### Что реализовано
- Пользователь выбирает CSV через кнопку «Загрузить CSV» в `DataSourcePanel`.
- Файл читается на клиенте, парсится утилитой `parseCsvFile`.
- Результат отображается в таблице предпросмотра панели данных.
- Дополнительно отображаются метаданные файла и ошибка (если есть).
- Данные ограничиваются по количеству строк для предпросмотра (`PREVIEW_LIMIT`), чтобы не перегружать UI.

### Задействованные файлы
- `src/utils/csv.ts` — универсальный парсер CSV
- `src/App.tsx` — orchestration загрузки, парсинга и состояния данных
- `src/components/DataSourcePanel.tsx` — UI загрузки/ошибок/предпросмотра
- `src/styles/globals.css` — стили статусов, ошибок и таблицы предпросмотра

### Текущий формат состояния данных (в `App.tsx`)
- `uploadedFileName?: string`
- `uploadedFileSizeBytes?: number`
- `csvPreviewRows: Array<Record<string, string | number | boolean | null>>`
- `csvTotalRows: number`
- `csvColumnCount: number`
- `csvParseError: string | null`

### Ограничения текущей реализации
- Поддерживается только CSV (JSON/XML запланированы дальше).
- Предпросмотр ограничен по количеству строк.
- Не реализованы стриминг/чанкинг для очень больших файлов (можно добавить в следующей итерации).

## Технические детали

### Структура компонентов (актуальная)

```text
src/
├── components/
│   ├── AppLayout.tsx          # Главный контейнер
│   ├── DataSourcePanel.tsx    # Левая панель + загрузка и предпросмотр CSV
│   ├── NodeCanvas.tsx         # Центральное поле
│   └── DashboardPanel.tsx     # Правая панель
├── hooks/
│   └── useSidebarToggle.ts    # Хук для управления сворачиванием
├── styles/
│   ├── variables.css          # CSS переменные
│   └── globals.css            # Базовые стили
└── utils/
    └── csv.ts                 # Парсинг CSV
```

### Примерная логика компоновки (CSS Grid)

```css
.app-layout {
  display: grid;
  grid-template-columns:
    [left-start] var(--left-panel-width) [left-end center-start]
    1fr [center-end right-start] var(--right-panel-width) [right-end];
  height: 100vh;
}

.app-layout.collapsed-left {
  grid-template-columns:
    [left-start] var(--panel-collapsed-width) [left-end center-start]
    1fr [center-end right-start] var(--right-panel-width) [right-end];
}
```

### Состояния компонентов
- `expanded` — панель открыта (видна полностью)
- `collapsed` — панель свернута (показывается иконка/ручка раскрытия)
- `focused-left` / `focused-right` — расширенный режим боковой панели (почти на весь экран)
